\doxysubsection{Detailed Description}
Singleton class for logging note and task events within the application. 

The \doxylink{class_logger}{Logger} class provides functionality to log various events related to notes and tasks, such as creation, deletion, updates, and status changes. It uses Qt\textquotesingle{}s QObject for signal-\/slot capabilities and supports event type enumeration for easy event identification.

Usage\+: \doxylink{class_logger_a4aa4d7c3b98f6e12e7ea8253da5ea0cd}{Logger\+::instance()}.log\+Event(Logger\+::\+NOTE\+\_\+\+CREATED, "{}\+Note\+Name"{});

\begin{DoxyNote}{Note}
This class follows the singleton pattern. Use \doxylink{class_logger_a4aa4d7c3b98f6e12e7ea8253da5ea0cd}{Logger\+::instance()} to access the logger. 
\end{DoxyNote}

This function prepares a JSON object containing the note name and, if provided, the task name. It then converts the event type to a string and calls the database manager to add the event log. The result of the operation is output to the debug log.
