\doxysubsection{Detailed Description}
Model class for managing a list of TODO notes in a Qt MVC application. 

This class inherits from QAbstract\+List\+Model and provides an interface for storing, retrieving, and manipulating TODO notes. It supports custom roles for note ID, item name, and timestamp, and exposes methods for adding, removing, and fetching notes.

\begin{DoxyNote}{Note}
The model uses a QVector of \doxylink{structnotes_element}{notes\+Element} to store its data.
\end{DoxyNote}
