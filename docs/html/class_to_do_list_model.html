Detailed Description

Model class for managing a list of to-do items in a Qt MVC application.

Inherits from QAbstractListModel and provides an interface for storing, retrieving, and manipulating to-do list items. Supports custom roles for item ID, name, and status. Exposes methods for adding, removing, toggling task status, and fetching data from a database. Integrates with Qt's meta-object system for use in QML and signal-slot communication.

Note
This class is intended to be used as the model in a Model-View-Controller (MVC) pattern, typically with a QListView or similar view component.

◆ fetchListFromDB()

void ToDoListModel::fetchListFromDB ( )

Fetches the to-do list items from the database and updates the model.

This function retrieves the list of to-do items associated with the current note ID from the database using DBManager. It resets the model, clears the existing data, and populates the model with the fetched items. Each item includes its ID, content, and completion status.

The model is reset before and after updating to ensure proper notification of views.
